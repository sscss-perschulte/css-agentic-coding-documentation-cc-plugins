Use the update-doc-agent to refresh documentation for: $ARGUMENTS

The agent will:
1. Find existing documentation related to this topic
2. Analyze what changed in the implementation
3. Update both operational and architectural docs (150-250 words each)
4. If no docs exist, create brief new ones

Workshop-optimized: Quick updates without massive rewrites.
